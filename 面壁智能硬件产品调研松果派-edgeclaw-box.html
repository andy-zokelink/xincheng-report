面壁智能硬件产品调研：松果派 & EdgeClaw Box

调研时间：2026年5月29日 · 数据源：秘塔AI搜索（Metaso MCP）

一、松果派（Pinea Pi）—— 面壁智能首款AI硬件

1.1 基本信息
维度 内容
发布 2026年2月官宣，预计2026年年中量产上市
定位 AI原生（AI Native）端侧智能开发板
理念 "Agent原生" + "自然语言直驱"
命名灵感 出自《三体》，寓意更高维度+全能力推动智能发展
目标用户 开发者、极客、AI硬件爱好者
开发门槛 无技术背景也可快速上手开发
1.2 硬件规格
维度 内容
硬件平台 NVIDIA Jetson系列模组
算力 275 TOPS（INT8）
传感器 内置麦克风、摄像头
接口 丰富的扩展接口（具体接口类型待量产确认）
电源 适配端侧低功耗场景
1.3 软件与模型
  • 预装模型：MiniCPM-o 4.5全模态大模型（语音/视频/文本同步交互）
  • 软件栈：系统优化 + 本地推理引擎 + 模型量化工具 + 智能中枢 + 开发者工具
  • 运行模式：全链路离线运行，数据不出设备，隐私安全，断网可用
  • 开发方式：自然语言直驱，降低传统嵌入式开发门槛
1.4 目标应用场景
场景 说明
离线多模态个人知识助理 本地处理文档/图片/语音，不依赖云端
具身智能 机器人/机械臂等硬件的端侧AI大脑
编程与教学 教育场景的AI交互开发平台
智能家居 本地AI中控，隐私安全
工业边缘计算 数据敏感场景的离线AI推理
1.5 总结定位

松果派是面壁智能"密度法则"从模型到硬件的落地。275 TOPS算力+MiniCPM全模态模型+全离线运行，对标NVIDIA Jetson生态但更强调AI原生和自然语言交互。目前尚未公布定价。

二、EdgeClaw Box（龙虾盒子）

2.1 基本信息
维度 内容
发布 2026年3月19日正式发布
定位 安全可控、开箱即用的"AI智能体一体机"
核心卖点 插电即用，一个盒子=一个数字员工团队
售价状态 企业版已开启预售，松果派版即将开放众筹
目标用户 OPC群体（非技术业务人员）+ 数据敏感型企业
发布方 面壁智能 + 清华大学 + OpenBMB社区 + 中国人民大学高瓴AI学院
2.2 核心架构

EdgeClaw框架（基于OpenClaw优化开源）：

特性 说明
基座 OpenClaw（开源智能体框架，昵称"龙虾"）
改进点 弥补OpenClaw在本地部署、数据安全上的不足
模型支持 主流云端模型 + 面壁MiniCPM端侧模型
运行模式 端云协同，支持离线运行
开源状态 联合清华大学、OpenBMB社区开源
2.3 隐私安全设计：隐私路由中间件

EdgeClaw Box最核心的技术创新——自研隐私路由中间件，实现信息分级处理：

数据类型 处理方式 模型 Token消耗
敏感数据 本地处理 MiniCPM系列离线运行 零Token消耗
非敏感数据 脱敏后调用云端 主流云端大模型 按需计费

双轨记忆机制：敏感记忆存本地，非敏感记忆可上云，防止隐私泄露。

2.6 总结定位

EdgeClaw Box不只是一个硬件，而是"硬件+框架+模型+安全"的一体化解决方案。它的核心价值在于降低AI智能体的部署门槛和数据安全顾虑，让企业无需技术团队也能拥有自己的私有AI员工团队。

四、对面壁智能战略布局的意义

面壁智能形成了"端侧模型（MiniCPM）→ 理论（密度法则）→ 硬件（松果派）→ 智能体框架（EdgeClaw）→ 开源社区（OpenBMB）"的全链路布局：

1. 松果派提供端侧AI推理的硬件底座 2. EdgeClaw Box提供即插即用的企业级智能体方案 3. MiniCPM系列提供轻量高效的端侧模型能力 4. 隐私路由中间件解决企业最关心的数据安全问题 5. 两条腿走路：开发者用松果派做原型，企业用EdgeClaw Box做落地

五、与中科领创的业务关联思考

1. 存储配套机会：松果派基于NVIDIA Jetson，康盈半导体/普冉半导体的存储芯片可作为配套方案切入 2. 边缘计算场景：中科领创的AIoT智能硬件方案 + EdgeClaw Box的边缘AI能力，可打包面向工业客户 3. 国产替代空间：松果派定价未公布，若有国产化替代需求，可关注是否有适配国产算力平台的版本 4. 本地化AI中控：EdgeClaw Box的"隐私AI智能体"概念，适合中科领创客户中对数据安全有高要求的场景（如军工/政务/金融）
