    一、Coze CLI 能力全景（八大模块）


    模块1：项目管理（coze code project）


    命令

    功能

    说明

    coze code project create

    创建项目

    自然语言描述需求，自动创建

    coze code project list

    查看项目

    列出所有项目

    coze code project delete

    删除项目

    清理不再需要的项目

    coze code deploy

    一键部署

    发布到云端，自动上线

    coze code preview

    预览链接

    生成在线预览地址


    价值：全程不用碰代码，说需求→自动创建→自动部署。适合快速做网页应用、小工具原型。


    模块2：多模态生成（coze generate）


    coze generate image "一只坐在云上的猫，水彩风格"     # 文生图
coze generate audio "你好，欢迎了解中科领创"          # TTS文字转语音
coze generate audio "系统播报" --output-path ./tts.mp3  # 指定输出
coze generate video create "芯片封装工艺流程展示"       # 文生视频
coze generate video create "日落延时" --wait --output ./sunset.mp4 --resolution 1080p --duration 8
coze generate video status <task_id>                   # 查询视频生成状态

    价值：生图/生视频/语音一行命令搞定，而且用的是字节自家模型（豆包+即梦+Seedance），效果有保障。


    模块3：对话与消息（coze code message）


    命令

    功能

    coze code message send

    发送消息给项目

    coze code message list

    查看消息历史

    coze code message cancel

    取消任务


    价值：相当于通过CLI跟你的Agent项目交互，适合自动化脚本里调用。


    模块4：技能管理（coze skill）


    命令

    功能

    coze skill add

    添加技能

    coze skill list

    查看已安装技能

    coze skill remove

    移除技能


    价值：365个技能商店工具随便装，覆盖法律/金融/自媒体/教育等。Agent装上技能就跟人装上专业牌照一样。


    模块5：环境变量（coze env）


    coze env set KEY=VALUE — 设置环境变量coze env list — 查看环境变量


    支持开发环境和生产环境区分。适合管理API Key、配置参数。


    模块6：空间管理（coze space / workspace）


    coze space list — 查看工作空间coze workspace switch — 切换工作空间


    模块7：文件管理（coze file）

    三、MCP生态——无限扩展的能力边界


    Coze的MCP支持分为三个层面：


    层面1：内置MCP插件


    扣子空间已支持自定义MCP插件，可直接在Coze平台内接入MCP Server。支持一键配置。


    层面2：coze-workflow-mcp-server


    开源项目，把你的Coze工作流暴露为MCP工具，其他Agent（如Claude Code、Cursor等）可以直接调用。安装方式：


    npx -y coze-workflow-mcp-server

    配置后，你写好的工作流就变成了可以被任何MCP客户端调用的工具。


    层面3：扣子与字节系生态打通


    火山引擎已集成近200种MCP生态服务，打通Trae/火山方舟/扣子三大平台。Coze 2.5的云电脑和云手机本身就是MCP服务的理想宿主。


    目前Coze可接入的能力类型：


  • 搜索类：实时联网搜索、谷歌/必应搜索· 图像类：文生图、图生图、图片编辑、QR生成· 视频类：文生视频、图生视频、数字人、视频编辑· 音频类：TTS、音乐生成、声音复刻、同声传译· 数据类：数据库查询、API调用、Excel处理· 办公类：飞书文档、多维表格、日历、邮箱· 行业类：法律、金融、教育、医疗专项插件



    • 四、Agent World——"Agent的社会"


    Coze 2.5最重磅的更新不是CLI本身，而是Agent World：


  • 每个Agent都有独立身份——有记忆、有工具、有社交关系· 每个Agent都有云电脑+云手机——可以自己操作浏览器、发邮件、运行脚本· 每个Agent都有独立邮箱——可以直接对外通信· Agent之间可以协作/竞争——多个Agent分工合作完成任务· 虾评平台：365个技能，Agent可以学习技能、上传技能，技能需经过众测才能上架


    • 这对你意味着什么：你可以创建多个Agent（如"客服小张"、"营销小王"、"财务小李"），每个Agent有独立的云环境、记忆和技能，7x24小时各自干活。你只需用Coze CLI下指令。

    💎 核心逻辑梳理


    本质问题


    Coze CLI不是"又一个命令行工具"，它是扣子2.5整个Agent生态的终端入口。它的价值不在CLI本身，而在它背后能调动的一切：365个技能、工作流引擎、云电脑/云手机、Agent World、MCP生态。


    驱动因素


    1. 字节系生态——扣子与飞书/豆包/火山引擎/抖音/Trae深度打通

    2. 你已是会员——零边际成本，不用就是浪费

    3. 门槛极低——自然语言驱动，说需求就干活

    4. Agent World——这个生态概念很前沿，Agent有身份/记忆/社交


    跟其他三家的关系


  • 扣子Coze = 主力（已付费，bot+工作流+多模态+生态覆盖70%需求）· 腾讯CloudBase = 做小程序/网站时补位（免费）· 火山veCLI = 做高质量视频/语音时补位（豆包原生的更专业）· 阿里百炼 = 查某个模型能力/价格时的查询入口
