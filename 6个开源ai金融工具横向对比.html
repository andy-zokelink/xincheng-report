交叉调研报告：6个开源AI金融工具横向对比

调研日期：2026-05-29 | 对象类型：工具/软件 | 调研人：阿爪

一句话结论

开给你的6个工具清单里，只有Vibe-Trading值得你现在装（装Hermes上做A股回测），其余5个要么不适合你的场景，要么你没法用。

各对象速览

1. HKUDS/AI-Trader (star 18.4k) · 比赛擂台，不是工具

核心数据：港大数据科学实验室 · 2025年10月开源定位：让5个AI模型各拿\$10,000在纳斯达克100里PK交易的比赛平台。实测DeepSeek收益率最高(+10.61%)，Claude第二(+4.03%)。结论：你用不上。它是研究"AI模型谁交易更强"用的，不是你做A股的工具。需要Alpha Vantage + Jina AI两套API Key。

2. Vibe-Trading (star 4k+) · 真正能用的零代码量化工作台 [推荐]

核心数据：港大实验室30天做出 · 2026年4月开源定位：自然语言驱动的AI多智能体金融工作台。你说"写一个MACD金叉回测"，它自动写代码、跑回测、出报告。核心能力：

  • 71个金融技能分7大类（数据源/策略/分析/资产/加密货币/资金流/工具）
  • 29个预设专家团队（投资委员会/量化策略组/风控委员会等）
  • 7种回测引擎 + 跨市场组合回测
  • 支持A股/港股/美股/加密货币/期货/外汇
  • AKShare免费数据开箱即用
  • 一键导出到TradingView/通达信/同花顺/MT5
  • pip install即可安装，支持Docker
  • 自带MCP Server可集成到Hermes风险：AI生成的策略底层逻辑可能不透明，用户需理解杠杆、回撤等概念。有案例显示回测亮眼的策略一年后实盘暴跌58%。结论：最值得装的。A股AKShare免费直连，自然语言驱动，不用写代码就能跑策略回测。配合DeepSeek V3.2成本极低。
3. TradingAgents (star 53k+) · 华尔街团队模拟器

核心数据：TauricResearch（清华+MIT背景） · 2024年12月开源定位：5.3万星标的超爆款。模拟华尔街交易团队——基本面分析师+情绪分析师+新闻分析师+技术分析师→多空辩论→交易员→风控→基金经理审批。核心能力：

  • 5层角色架构：分析师团队(4人) → 研究团队(辩论) → 交易员 → 风控 → 基金经理
  • 基于LangGraph多智能体编排
  • 苹果股票模拟交易实现+26.6%回报门槛：需要FinnHub API Key + OpenAI API Key。框架API调用量大，成本不低。A股不直接支持。结论：研究价值高，但门槛高。不适合你的日常A股场景。
4. Dexter (star 9.8k) · 财报深度研究Agent

核心数据：virattt · TypeScript+Bun运行时定位："金融领域的Claude Code"——自动分解复杂金融问题→规划→执行→自我验证。核心创新是LLM-as-judge自我纠错。核心能力：

  • 多Agent架构：规划Agent→执行Agent→验证Agent→回答Agent
  • 自动检查工作质量，不满足条件继续迭代
  • 数据源：Financial Datasets API门槛：Bun运行时Windows打包麻烦。数据源Financial Datasets API以美股为主，A股覆盖不明确。需要OpenAI + Financial Datasets + Exa三套API Key。结论：美股财报研究工具，对A股场景不适用。
5. anthropics/financial-services · 华尔街投行工具包

核心数据：Anthropic官方 · 2026年5月发布 · 10个Agent + 41个技能定位：为投行/私募/股权研究/财富管理/基金行政设计的AI Agent工具集。核心能力：

  • 10个Agent：Pitch Agent、Market Researcher、Earnings Reviewer、Model Builder、GL Reconciler、KYC Screener等
  • 11个专业数据源连接器（Daloopa/Morningstar/S&P Global/FactSet/Moody's/LSEG/PitchBook等，每个需单独付费订阅）
  • 两种部署：Claude Cowork插件或Managed Agents API结论：不装。华尔街投行专用，数据源全是机构级付费服务，A股完全不支持。需要Claude生态。
6. anthropics/knowledge-work-plugins (star 17.8k) · 通用知识工作者插件

核心数据：Anthropic官方 · 11个插件定位：Claude Cowork/Code的插件包。里面有个finance插件能做账目对账/财务报表/审计支持。核心能力：

  • 11个插件：productivity/sales/customer-support/product-management/marketing/legal/finance/data等
  • finance插件：记账分录、账目对账、财务报表、差异分析、月度结账、审计支持
  • 连接Snowflake/Databricks/BigQuery等企业数据仓库结论：需要Claude Code才能装。如果你在用Claude Code，finance和data插件值得装来对接金蝶ERP做财务分析。Hermes不兼容。

综合判断

只推荐装一个：Vibe-Trading。理由：

  • A股数据源开箱即用（AKShare免费，tushare可选）
  • 自然语言驱动，你不用写代码
  • pip install 5分钟，Hermes可以直接当MCP工具用
  • 跟DeepSeek V3.2搭配成本极低（官方推荐配置）
  • 回测后能导出到通达信/同花顺格式，接A股券商没问题

其他5个不装的原因：

  • AI-Trader：比赛平台，你不需要知道AI之间谁交易更强
  • TradingAgents：研究价值高但门槛高（FinnHub+OpenAI Key），A股不直连，成本不便宜
  • Dexter：美股财报分析，Bun运行时Windows打包麻烦，Financial Datasets API不确认支持A股
  • anthropics/financial-services：华尔街投行专用，每个数据连接器都要付费订阅
  • anthropics/knowledge-work-plugins：Claude Code专属插件，你没有Claude Code的话用不了

  • 各工具数据均来自公开信源，不构成投资建议
