🧠 技术路线：什么是"认知模型"？

大白话解释

传统大模型（DeepSeek、Qwen、Kimi）像是一座超级图书馆——把所有书都背下来，你问什么它从记忆里找答案。

新程 Alpha 像一个聪明人——不背书，但知道怎么思考、怎么推理、怎么查资料、怎么知道自己不知道什么。图书馆越大越重，而聪明人只需要一个好脑子。

🔧 技术原理

  • 知识-认知解耦：把模型的"知识储备"和"思考能力"拆开，剥离死记硬背的部分，只保留纯推理核心
  • 强化学习驱动：对开源推理模型做大规模 RL 训练，让模型学会"怎么想"而不是"记什么"
  • 群体智能研究：梳理了 1800-2020 年共 220 年人类学术论文，归纳出群体智能的演化规律，指导模型设计
  • 泛化优先：不追求在特定数据集上刷分，追求跨领域、跨任务的抽象推理能力

🚀 怎么部署和使用？

当前状态 模型权重未开源，只能通过平台使用

  • HuggingFace 和 GitHub 上目前搜不到新程 Alpha 的模型权重，不能用 Ollama 拉、不能本地下载自己跑
  • 李笛在 2026 年 2 月专访中表示："未来几个月可能会开源部分认知模型相关工作"，但"不会走数量很多的路线"
  • 团队训练方法基于开源推理模型做强化学习，技术上开源可行——只是时间问题

🌐 团子平台（当前唯一使用方式）

网址：https://mytuanzi.com/

Nextie 的多智能体平台，新程 Alpha 作为底层认知模型驱动其中的智能体。工作机制是：用户提问 → 多个智能体辩论/反思/投票 → 达成共识 → 输出结论。不是单模型给答案，是群体辩论后收敛出结论。

  • 姐妹团——偏个人生活理解、情感问题，多视角共情分析
  • 研究团——对标 Deep Research，跨学科深入分析复杂问题
  • 问奇绩——创业咨询，模拟投委会质询（与奇绩创坛合作）

平台按周迭代（每周三更新），持续上线新团/新角色/新能力。近期计划：专业写作团、千人以上群体仿真团。

🔌 API 和商业化

  • 已实现商业化部署（2026 年 6 月确认）
  • 目前没有公开 API，只能通过团子网页端使用
  • 李笛表示未来会开放用户自定义更换底层模型——团队定位是做"Agent OS / 平台层"，API 化是确定性方向
  • 底层认知模型支持端侧部署（MacBook、机器人等设备原生运行）
⚠️ 当前实操结论：想用 → 去 mytuanzi.com 体验团子；想本地部署 → 等开源。

🖥️ 实际硬件推荐（假设未来开源）

Mac 用户：

  • 16GB 统一内存 → 流畅运行（可同时跑系统和模型）
  • 8GB 也有可能，需 INT4 量化版本
  • M 系列芯片对推理优化好，速度通常比同级别 NVIDIA 卡更快

Windows / Linux 用户：

  • 8GB 显存 GPU（如 RTX 3060/4060）绰绰有余
  • 纯 CPU 也能跑（16GB 内存以上），只是速度慢
  • 对比参考：一般 7B 模型需约 14-16GB 显存（FP16），4B 只需一半

一句话：硬件门槛极低，普通笔记本就能跑。

📱 方向三：端侧部署国产小模型

  • 面壁智能 MiniCPM 系列——国产端侧第一梯队，All in 端侧。"大模型摩尔定律"：知识密度每 8 个月翻番。MiniCPM 3.0（4B）→ 4.0（8B/0.5B）→ MiniCPM-o 4.5（9B 多模态）→ MiniCPM5-1B。下载量破千万。与新程 Alpha 定位差异：面壁做通用端侧智能（对话/视觉/语音全覆盖），不专注认知模型
  • 商汤 SenseChat-Lite——手机/IoT 端侧方案，"模型+芯片适配"一体化，安防/车载有落地。参数估计 1-4B。走行业解决方案路线，不追求认知泛化
  • RockAI（岩芯数智）——同样押注"群体智能"概念，2024 年 9 月发声，国产群体智能赛道最早玩家之一。但未见与 Nextie 同级别的产品化深度和 benchmark 数据
  • 广和通「星云」系列——通信模组厂商，非模型公司，整合外部模型做"硬件+模型打包"交付，支持 7B 及轻量级小模型，覆盖 1T-50T 算力

🔑 核心结论

新程 Alpha 在认知模型这个品类里目前没有直接竞品——其他 4B 模型做的是"小号百科全书"，它做的是"思考引擎"。但问题在于：

  • 可替代性强——如果你只需要一个 4B 端侧模型处理日常任务，MiniCPM / Phi / Qwen 更成熟（开源、有文档、社区活跃）
  • 护城河待验证——"4B 打平 GPT-5.4"的数据来自 Nextie 自建的 IDI 评测，缺少第三方独立复现
  • 框架层威胁——Isara+GPT-5.4 的组合可能从框架侧绕过认知模型的壁垒
  • 最大变数——如果认知模型的效果能在独立 benchmark 上复现，品类的壁垒就成立；如果不能，就只是一个经过特定场景 RL 微调的小模型

💡 核心判断

DeepSeek / Qwen / Kimi 比的是"谁知道得多"，新程 Alpha 比的是"谁会思考"。

用 4B 参数做到千亿模型的多智能体协作效果，把 AI 从"烧显卡"变成"交电费"，押注的是群体智能 + 端侧部署这两个下一代方向。

风险提示：目前公开 benchmarks 集中在群体智能赛道，通用知识/编程/创作能力尚未见横向对比数据。认知模型路径是否正确，取决于规模化后的表现。
